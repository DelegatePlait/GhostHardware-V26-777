Get detailed insights into your computer's hardware instantly.

**PC Hardware Diagnostic Tool** is a fast, lightweight, and completely safe utility designed for Windows users who want to analyze their system configuration. Whether you are upgrading your PC, checking part compatibility, or retrieving serial numbers for tech support, this tool provides all the data you need in one clean interface.

---

### 🚀 Key Features

* **🔍 Deep System Scan:** Retrieve detailed information about your CPU, GPU, RAM, Motherboard, and Storage.
* **📋 Serial Number Viewer:** Easily view component serial numbers and HWID strings for diagnostics and asset tracking.
* **💻 Windows Native:** Optimized to run smoothly on Windows 10/11 without heavy dependencies.
* **📊 Exportable Reports:** Save your hardware configuration as a clean text or HTML report with one click.
* **🛡️ Safe & Read-Only:** This utility only reads system information. It does not modify any hardware registry keys or system files.
